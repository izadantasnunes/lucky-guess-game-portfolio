# Lucky Guess Game

Jogo de adivinhação com visual premium em verde neon, interface responsiva e interações em JavaScript puro.

## Estrutura
- `index.html`
- `css/style.css`
- `js/script.js`

## Features
- 10 tentativas
- feedback de palpite maior/menor
- popup de vitória e derrota
- efeito visual premium
- layout responsivo

## Publicação
Projeto pronto para VS Code, GitHub e GitHub Pages.
